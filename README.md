## [Vue项目](./project)

### 🔥🔥建立项目

#### 1. 使用 vite-app

```shell
npm init vite-app <project-name>
```

这里的vite-app是一个新项目，它的官方介绍是一个快速的WEB开发构建工具。这里我们试了一下，整个构建过程十分的快速。和以往的webpack build的方式不一样，它使用了原生ES模块加载。

#### 2. 使用vue-cli

```shell
npm install -g @vue/cli # OR yarn global add @vue/cli
vue create <project-name>
```
